Design Notes

  • Close button: Positioned at the far right, uses Material Symbols "close" icon
  • Hover state: Subtle background highlight on hover, similar to other close buttons in the app
  • Spacing: Maintains consistent 16px gap between elements
  • Flexibility: Can be used with or without the Undo button depending on context
  • Visual hierarchy: Close button is more subtle (secondary color) compared to the action button (blue)
  • Message behavior: Message text uses flex: 1 to fill available space and wrap naturally
