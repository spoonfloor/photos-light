Key Design Decisions:

1. 48√ó48px thumbnails - Large enough to identify content, small enough to not dominate the row
2. Folders have no thumbnails - Keeps folder vs file distinction clear
3. Shimmer loading state - Better than spinners, feels more modern
4. Metadata on second line - Dimensions and size help identify files
5. Error state is subtle - Warning icon but doesn't block interaction
6. Lazy loading - Only generate thumbnails for visible items + 100px buffer
